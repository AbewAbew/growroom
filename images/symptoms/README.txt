NOTE: The Diagnose page now loads its reference photos from the
project-root folder  scraped_symptoms/  (mapped per symptom in
js/diagnose.js via each entry's "img" field).

This images/symptoms/ folder is no longer used by the app. To change a
symptom's photo, drop your file in scraped_symptoms/ and set that
symptom's "img" to the filename in js/diagnose.js.

Symptoms without a photo in the set (spider mites, fungus gnats,
aphids/whiteflies) show a drawn leaf illustration until you add one.
